shoutpage
=========

Run this on the same server as Shout-IRC.

Envvars
-------

| Name          | Meaning                            |
|:------------- |:---------------------------------- |
| `ATHEME_URL`  | URL for Atheme client to log into  |
| `BNC_URL`     | URL for BNC clients to log into    |
| `OUTPUT_PATH` | Where to write shout json files to |
